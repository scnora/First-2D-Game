# Save the King 

Hi everyone! This is my first ever 2D game and I hope you enjoy! This game was developed by me with the help of AlexDev on Youtube! His courses are very insightful and have definitely been so crucial in my game dev journey.
Despite this game being an example he used during his course, I found it very exciting and challenging to recreate! I've learned so much skills within game dev like Polymorphism, Inheritance, Game UI, and more!
